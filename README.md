# Cálculo de Expensas por Departamento

Proyecto hecho con HTML, CSS y JavaScript que permite registrar departamentos, asignarles metros cuadrados y calcular categorías de expensas.

## Funcionalidades
- Agregar departamentos
- Guardar metros cuadrados
- Clasificar por categorías según tamaño
- Visualización en tabla

## Tecnologías usadas
- HTML
- CSS
- JavaScript

## Objetivo
Práctica de manipulación del DOM y lógica en JavaScript para gestión de datos en una interfaz web.
